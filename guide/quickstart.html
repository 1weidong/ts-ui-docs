快速开始 ​

本文档将帮助你快速集成并使用 ts-ui 组件库。

1. 安装组件库 ​

使用 pnpm 安装 ts-ui：

bash
pnpm add ts-ui

如果你还未安装 pnpm，可以先通过 npm 全局安装 pnpm：

bash
npm install -g pnpm

2. 在 Vue 3 项目中引入组件库 ​

在你的 Vue 3 项目中，通过以下步骤引入 ts-ui：

2.1 全局注册 ​

在 main.ts 中引入并注册组件库，这样你可以在项目中直接使用各个组件：

ts
import { createApp } from 'vue'
import App from './App.vue'
import tsUI from 'ts-ui' // 引入组件库

const app = createApp(App)
app.use(tsUI) // 全局注册所有组件
app.mount('#app')

2.2 局部注册 ​

如果你只想在部分页面或组件中使用 ts-ui 的某个组件，也可以直接按需导入：

ts
import { defineComponent } from 'vue'
import { Button } from 'ts-ui' // 导入 Button 组件

export default defineComponent({
  name: 'MyComponent',
  components: {
    TsButton: Button // 注册为局部组件
  },
  template: `
    <div>
      <TsButton>点击我</TsButton>
    </div>
  `
})

3. 运行示例 demo ​

为了便于开发调试，建议你在组件库项目中使用 demo 应用进行预览。

3.1 安装 demo 依赖 ​

进入 demo 目录并安装依赖：

bash
cd demo
pnpm install

3.2 启动开发服务器 ​

在 demo 目录下启动 Vite 开发服务器：

bash
pnpm run dev

此时，打开浏览器访问 http://localhost:5174 即可看到 demo 页面，并测试各个组件的效果。

4. 编写单元测试（可选） ​

为了保证组件的稳定性，建议为每个组件编写单元测试。示例：为 Button 组件编写测试：

ts
// tests/Button.spec.ts
import { mount } from '@vue/test-utils'
import { describe, it, expect } from 'vitest'
import Button from 'ts-ui/components/Button'

describe('Button 组件', () => {
  it('正确渲染 slot 内容', () => {
    const wrapper = mount(Button, {
      slots: {
        default: '点击我'
      }
    })
    expect(wrapper.text()).toContain('点击我')
  })
})

运行测试：

bash
pnpm test

通过以上步骤，你可以迅速上手 ts-ui 组件库，并在项目中调用各个组件进行开发和调试。随着项目的不断完善，还可以根据实际需求扩展更多组件和功能。
