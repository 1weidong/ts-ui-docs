介绍 ​

ts-ui 是一个基于 Vue 3 开发的高质量组件库，旨在帮助开发者快速构建现代化、响应式和可定制的 Web 应用。该组件库采用 Vite 作为构建工具，利用 TypeScript 保证代码质量与类型安全，并结合了 Tailwind CSS 和 SCSS 以实现灵活且高效的样式管理。

项目亮点 ​

Vue 3 & Vite ​

使用最新的 Vue 3 特性和 Vite 快速构建，享受极速热更新与高效打包体验。

TypeScript 支持 ​

全面使用 TypeScript 进行开发，提供完善的类型声明，帮助开发者捕获潜在错误，提高代码可维护性。

按需导入 ​

通过配置插件实现组件的按需导入，减少打包体积，提升加载速度。

灵活的样式体系 ​

集成 Tailwind CSS 和 SCSS，让你在享受工具化样式的同时，也可以自由定制主题和样式。

丰富的组件文档 ​

使用 VitePress 构建文档站点，包含完整的侧边栏和导航，详细说明各组件的使用方法和 API，助你快速上手。

完善的单元测试 ​

利用 Vitest 和 @vue/test-utils 实现单元测试，确保每个组件在各种场景下都能正常工作。

适用场景 ​

打通design tool和 tapstitch 的交互规范。

自定义组件库，满足特殊业务需求，同时享受开源生态带来的便利。
